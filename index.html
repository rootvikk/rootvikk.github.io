01 — about

Player One.

I'm Viraj — a first-year B.Tech Computer Science student at heart and a game developer by obsession. I started making games because I wanted to understand how the worlds I played in actually worked.

My workflow lives in Unity and C# — scripting mechanics, building systems, and sweating the small stuff until a game feels right to play. I use Blender for 3D assets when I need something custom, and Git + GitHub to keep everything sane.

I've shipped one game on itch.io and I'm building the next. Every project teaches me something new about the craft — and I'm just getting started.
